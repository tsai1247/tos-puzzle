## 專案基本架構
    - Github Page 一頁式，純前端
    - 使用熱門套件來做設計，不要純html，建議 react + vite
    - local 測試時路徑放在 port 2178
## 規則
- 有編輯階段與拼圖階段
### 版面統一規範
    - 以左下角的方格為 (0, 0), 向右是 X 軸, 向上是 Y 軸
### 編輯階段
    - 可以選擇 9*9, 12*12, 16*16 或 20*20 的場地
    - 有四種工具：單格選中、單格取消選取、方形選中、方形取消選取
        - 單格就是使用者滑鼠點擊拖曳的路徑都變成選中/取消選取狀態
        - 方形就是使用者滑鼠點擊兩格，形成的方格都變成選中/取消選取狀態
        - 選中的意思就是後續可以拼圖的目標區塊
        - 取消選取的意思就是後續雖然可以放拼圖，但不是目標，會稍微扣分的區塊
    - 有儲存功能，放在 localstorage 
    - 有讀取功能，可以讀取 localstorage 或預設版面(由我後續提供)
    - 有匯出功能，將當前場地與版面簡單 encode 後匯出成檔案
    - 有匯入功能，將檔案 decode 後覆蓋當前場地與版面

### 拼圖階段
#### 基本規則
    - 有五種屬性的拼圖，顏色分別為 藍色、紅色、綠色、黃色、紫色
        - 每個屬性的拼圖都有以下四種形狀
            1. 5格組成的T字型，具體形狀是(0, 2), (1, 2), (2, 2), (1, 1), (1, 0)
            2. 5格組成的十字型，具體形狀是(0, 1), (1, 2), (2, 1), (1, 1), (1, 0)
            3. 5格組成的L字型，具體形狀是(0, 0), (0, 1), (0, 2), (1, 2), (2, 2)
            4. 4格組成的一字型，具體形狀是(0, 1), (1, 1), (2, 1), (3, 1)
        - 每個屬性的拼圖都另外有一個 "大型7格拼圖"
            1. 藍色：(1, 0), (1, 1), (0, 2), (1, 2), (2, 2), (0, 3), (2, 3)
            2. 紅色：(0, 0), (0, 1), (0, 2), (1, 1), (1, 2), (2, 2), (3, 2)
            3. 綠色：(0, 0), (0, 1), (0, 2), (1, 1), (2, 0), (2, 1), (2, 2)
            4. 黃色：(0, 0), (0, 1), (0, 2), (1, 2), (2, 0), (2, 1), (2, 2)
            5. 紫色：(0, 0), (0, 1), (0, 2), (1, 0), (1, 1), (1, 2), (2, 0)
    - 有萬能拼圖，分別是 1*1, 1*2, 1*3 的灰色方塊
    - 每個拼圖都可以旋轉，但不可以翻轉
    - 拼圖不可重疊
    - 所有拼圖數量不限
#### 模式
    - 手動模式，使用者選取拼圖，以點選、拖曳的方式完成拼圖，拼圖被點選時一旁會有刪除與旋轉按鈕
    - 自動模式，使用者選取條件，程式 greedy 演算法拼出所有完美組合
        - 完美組合的意思是，僅拼完所有目標區塊，不拼到扣分區塊
        - 結果記錄前10種高分可能性，排序為：使用越少萬能拼圖越高分，萬能拼圖數量一樣的話以使用越少7格拼圖的越高分
        - 使用者可以點擊紀錄的可能性來套用到版面上查看
        - 每個屬性的4種共同形狀，只使用藍色拼圖，因為不同顏色不影響拼法，會太多重複
        - 可選擇條件(複選)有：
            - 可使用基本形狀
            - 可使用大型7格拼圖
            - 可使用萬能拼圖
        - 至少要選一個條件，最多可以三個條件都選
    - 右上角總是顯示數值：已拼方格數量、未拼方格數量、超出方格數量
#### 美化
    - 每個拼圖都需要注意外框的視覺感，避免多個相同拼圖密集合在一起時，看不出邊界
    - 加上 "隨機換色" 按鈕，會將版面上四種共同拼圖隨機換成藍色、紅色、綠色、黃色、紫色
#### 儲存
    - 儲存按鈕可以將當前拼圖解法儲存為 "圖片" 或 "encode檔案"
        - encode檔案包含場地大小、版面與拼法
    - 讀取按鈕可以讀取 encode檔案，覆蓋當前場地、版面、拼法
        - 場地大小或版面與當前不符合時，跳出警告是否覆蓋
